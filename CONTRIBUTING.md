# Contributing

Obrigado pelo interesse em contribuir para este projeto!

## Como contribuir

1. Faz **fork** do repositório
2. Cria uma **branch** para a tua feature (`git checkout -b feature/nome-da-feature`)
3. Faz **commit** das tuas alterações (`git commit -m "Adiciona nova feature"`)
4. Faz **push** para a branch (`git push origin feature/nome-da-feature`)
5. Abre um **Pull Request**

## Setup local

```bash
bun install
```

Certifica-te de que tens o ChromaDB a correr localmente e as variáveis de ambiente configuradas (ver README.md).

## Regras

- Escreve código em **TypeScript** com strict mode
- Segue as convenções já existentes no projeto
- Texto visível ao utilizador deve estar em **português europeu**
- Testa as tuas alterações antes de submeter o PR

## Reportar bugs

Abre uma [issue](https://github.com/hvasconcelos/constituicao/issues) com:
- Descrição do problema
- Passos para reproduzir
- Comportamento esperado vs. actual

## Sugestões

Ideias e sugestões são bem-vindas! Abre uma issue com a tag `enhancement`.
